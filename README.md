# Aether Weather — Professional Weather Dashboard

A modern, responsive, and highly customizable weather forecasting dashboard featuring a glassmorphic ("liquid glass") volumetric design, dynamic 3D-tilt effects, interactive audio feedback, and worldwide location search. Developed and polished with the assistance of **Google AI**.

## 🌟 Key Features
- **3D Tilt Effect**: Volumetric and fluid interactive movement of the main card, reacting smoothly to mouse tracking.
- **Audio Feedback**: Custom click sounds with a dedicated volume controller.
- **Worldwide Geocoding**: Search and filter any city globally on the fly with automatic timezone synchronization.
- **Extensive Customizable Settings**:
  - Multiple unit types (Celsius/Fahrenheit, m/s or km/h, mmHg or hPa).
  - Multi-theme support (Glass, Cyberpunk, Aurora, Sunset, Dark, Light).
  - Card opacity, blur density, and background blob animation speed controls.
- **Dynamic Astro Arc**: Physical calculation and visual tracking of the sun's trajectory according to the local timezone.

## 🛠️ Technological Stack
- HTML5 / CSS3 (Modern Glassmorphism & Custom Properties)
- Vanilla JavaScript (No heavy frameworks, highly optimized DOM manipulation)
- [Open-Meteo API](https://open-meteo.com) for real-time and 10-day weather forecast data
- [Open-Meteo Geocoding API](https://open-meteo.com) for worldwide city search

# Aether Weather — Профессиональный погодный дашборд

Современный, адаптивный и кастомизируемый дашборд для прогнозирования погоды с объемным дизайном «жидкого стекла» (Glassmorphism), динамическим 3D-наклоном, интерактивными звуковыми эффектами и глобальным поиском по всему миру. Разработано и оптимизировано при поддержке **Google AI**.

## 🌟 Основные возможности
- **Интерактивный 3D-наклон**: Объемная левитация главной карточки, плавно реагирующая на ведение курсора.
- **Звуковое сопровождение**: Звуковые клики при нажатии кнопок с полным контролем громкости.
- **Глобальный поиск**: Быстрый поиск любого города на любом языке с автоматической синхронизацией локального времени.
- **Глубокая кастомизация настроек**:
  - Изменение единиц измерения (°C/°F, м/с или км/ч, мм рт. ст. или гПа).
  - Поддержка 6 уникальных тем (Стекло, Киберпанк, Аврора, Закат, Темная, Светлая).
  - Регулировка прозрачности стекла и скорости движения фоновых сфер.
- **Астрономическая дуга**: Физический расчет траектории солнца в зависимости от текущего локального времени.

## 🛠️ Технологический стек
- HTML5 / CSS3 (Современный глассморфизм и CSS-переменные)
- Чистый JavaScript (Без тяжелых фреймворков, оптимизированная работа с DOM)
- [Open-Meteo API](https://open-meteo.com) для получения точных метеоданных
- [Open-Meteo Geocoding API](https://open-meteo.com) для поиска городов мира
